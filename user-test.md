# User test

Ask a learner to complete this task without explaining the interface:

> Compare the default moving-average strategy with buy-and-hold. Identify which
> performed better in this fixed sample, then explain two reasons the result
> should not be used as a trade recommendation.

Record:

- whether the learner found the source cards;
- whether the learner could run the backtest;
- whether the learner distinguished historical result from future performance;
- where the learner hesitated or misunderstood the interface.

Passing requires task completion and two valid limitations. A profitable
strategy is not a passing condition.
