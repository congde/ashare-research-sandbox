# Lab 04: research report

This lab demonstrates a research deliverable with traceable sources. The
starter report reads well but fails verification because facts are not tied to
sources. The solution separates facts, inferences, recommendations, and
unknowns.

## Ticket

Read [ticket.md](ticket.md), then compare `starter/research-report.md` with
`solution/research-report.md`.

## Verify the teaching fixture

```bash
make lab-04
```

## Verify your own report

```bash
.venv/bin/python labs/04-research/verify.py path/to/your-report.md
```
