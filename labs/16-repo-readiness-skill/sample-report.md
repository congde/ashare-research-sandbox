# Repository readiness

## Purpose

`ashare-research-sandbox` is a companion workspace for a Codex delivery course. This is
confirmed by `README.md` and the publishable drafts under `docs/v2/`.

## Entrypoints

- `AGENTS.md`: repository instructions for Codex.
- `Makefile`: setup and verification commands.
- `labs/`: executable teaching fixtures.
- `skills/`: reusable Codex workflows.

## Verification

Run `make check` to verify all executable teaching fixtures.

## Risks

Course prose can drift away from the files and commands it names.

## Unknowns

Publication platform formatting requirements are not stored in this repository.
